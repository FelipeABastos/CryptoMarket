# CryptoMarket

This app consists in a list of exchanges, containing some informations inside a detail screen.

## Requirements

```
- iOS 13+
- Xcode 13
- Swift 5
```

## This project uses:

```
- UIKit
- VIPER
- View Code
- URL Session
- Dependency Injection

✅ XCTest: Unit & UI
✅ SwiftUI: XCode Canvas (For previewing the UI in realtime)
```

## Swift Package Manager
```
- Kingfisher
- Lottie
- Notification Banner
```
